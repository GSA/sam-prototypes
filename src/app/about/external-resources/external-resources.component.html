External Resources

Entity Resources

  • CAGE Code Search. The Defense Logistics Agency (DLA) Commercial and Government Entity (CAGE) portal to search for domestic and foreign entity CAGE Code information.
  • DUNS Number Request. The webform where Dun & Bradstreet (D&B) provides a DUNS Number, a unique nine digit identification number, for each physical location of your business. DUNS Number assignment and maintenance is free for all entities required to register with the U.S. government for contracts or grants.
  • NCAGE Code Request. The online portal for international entities to request a new NATO CAGE (NCAGE) Code or submit updates to an existing NCAGE Code record.
  • Procurement Technical Assistance Centers (PTACs). An official resource for government contracting assistance. Find your local PTAC to get free help completing your SAM registration.
  • SBA's Dynamic Small Business Search (DSBS). The searchable tool which displays SAM entity registration data and supplemental information provided by small businesses to SBA to further describe their entity.
  • U.S. Census Bureau. The official source for data about the nation's people and economy including the North American Industry Classification System (NAICS) and Metropolitan Statistical Areas (MSA).
